# 🍫 Smart Expense Analyzer

## 📌 Description

Smart Expense Analyzer is a Python GUI application that helps users track their expenses, manage budgets, and analyze spending patterns. It also provides AI-based suggestions to improve saving habits.

---

## ✨ Features

* 💰 Add and categorize expenses
* 📊 Pie chart for expense distribution
* 📉 Bar graph for category-wise spending
* 📈 Trend graph for expense tracking
* 🧠 AI-based budget suggestions
* 📜 Expense history tracking
* 🧹 Clear all data option

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* Matplotlib (Graphs)

---

## 🚀 How to Run

### Step 1: Install Required Libraries

```bash
pip install matplotlib
```

### Step 2: Run the Program

```bash
python your_file_name.py
```

---

## 🧠 How It Works

1. User sets a budget
2. Adds expenses with categories
3. System calculates:

   * Total expenses
   * Remaining savings
4. AI gives suggestions like:

   * Budget exceeded
   * Moderate spending
   * Good savings
5. Graphs visualize the data

---

## 📊 Graphs Included

* Pie Chart → Expense distribution
* Bar Graph → Category-wise spending
* Trend Graph → Expense pattern over time
